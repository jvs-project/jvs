# Changelog

All notable changes to the JVS VS Code extension will be documented in this file.

## [0.1.0] - 2024-02-23

### Added
- Initial release of JVS VS Code extension
- Create snapshots with notes and tags
- Restore snapshots from command palette or tree view
- View snapshot history in sidebar
- List and manage worktrees
- Fork new worktrees
- Verify all snapshots
- Return to HEAD state
- Status bar integration showing snapshot count
- File system watcher for auto-refresh
- Context menus for quick snapshot actions
- Configuration options for history limit, default engine, and executable path
- Output channel for debugging and operation logs
